① Types de signaux

Signal Analogique

Continu en temps ET en valeur. Le signal naturel "brut".

On ne prélève la valeur qu'aux instants nT_e. Le temps devient discret.

Temps : discret Valeurs : continues

Échantillonné ET quantifié. Stockable en mémoire.

Temps : discret Valeurs : discrètes
✅ À RETENIR POUR LE QCM

Signal échantillonné = signal à temps discret (pas valeurs discrètes !)

Signal quantifié = signal à valeurs discrètes

Signal numérisé = échantillonné + quantifié (les deux !)

Un filtre numérique reçoit en entrée un signal numérisé.

② Peigne de Dirac et ses opérations

Définition du peigne de Dirac

Un peigne de Dirac de période T_e est une somme infinie d'impulsions de Dirac :

P_{T_e}(t) = Σ δ(t - n·T_e) pour n ∈ ℤ

Les 4 opérations fondamentales

🔴 Multiplication par δ(t - t₀)

x(t) × δ(t - t₀) = x(t₀) · δ(t - t₀)

Rôle : Échantillonnage — prélève la valeur de x en t₀

🔵 Convolution avec δ(t - t₀)

x(t) * δ(t - t₀) = x(t - t₀)

Rôle : Retardement — décale x(t) de t₀

🟡 Multiplication par P_Te(t)

x(t) × P_{T_e}(t) = Σ x(nT_e)·δ(t - nT_e)

Rôle : Échantillonnage périodique — prélève x tous les T_e

🟢 Convolution avec P_Te(t)

x(t) * P_{T_e}(t) = Σ x(t - nT_e)

Rôle : Périodisation — crée des copies de x espacées de T_e

TF d'un signal échantillonné

Si x(t) est échantillonné à la fréquence F_e = 1/T_e, sa TF est périodique de période F_e :

X_e(f) = (1/T_e) · Σ X(f - k/T_e) // répétitions espacées de F_e = 1/T_e
La TF d'un signal T_e-échantillonné est (1/T_e)-périodique en fréquence (pas T_e-périodique !)

⑤ Transformée en Z

Définition

X(z) = Σ x(n) · z^{-n} pour n allant de 0 à +∞

La variable z est analogue à e^{pT_e} dans le domaine de Laplace discret.

Table de Transformées en Z

δ(n)1

Méthode : Décomposition en éléments simples (Z)

Astuce : On ne décompose pas X(z) directement, mais X(z)/z = S(z)

S(z) = X(z)/z = A/(z−z₁) + B/(z−z₂) → X(z) = Az/(z−z₁) + Bz/(z−z₂) → x(n) = A·z₁ⁿ·u(n) + B·z₂ⁿ·u(n)
Coefficients : A = [(z−z₁)·S(z)]|_{z=z₁} B = [(z−z₂)·S(z)]|_{z=z₂}

ConvolutionZ[x*y] = X(z)·Y(z)

⑥ Filtres Numériques

Filtre RIF (non récursif)

La sortie ne dépend que des entrées passées

y(n) = a₀x(n) + a₁x(n-1) + a₂x(n-2)

✅ Toujours stable RIF

Réponse impulsionnelle finie

Filtre RII (récursif)

La sortie dépend des entrées ET des sorties passées

y(n) = x(n) + b₁y(n-1) + b₂y(n-2)

⚠️ Stable si pôles dans cercle RII

Réponse impulsionnelle infinie

Lire un bloc diagramme

Du bloc diagramme à H(z) — 3 étapes

Étape 1 : Lire l'équation aux différences y(n) = x(n) + 0.5·x(n-1) + 0.75·y(n-1) Étape 2 : Prendre la Transformée en Z Y(z) = X(z) + 0.5z⁻¹X(z) + 0.75z⁻¹Y(z) Y(z)[1 - 0.75z⁻¹] = X(z)[1 + 0.5z⁻¹] Étape 3 : H(z) = Y(z)/X(z) H(z) = (1 + 0.5z⁻¹) / (1 - 0.75z⁻¹)

⑦ Stabilité des Filtres Numériques

📐 CONDITION DE STABILITÉ

Un filtre numérique RII est stable si et seulement si tous ses pôles sont situés à l'intérieur du cercle unité dans le plan complexe Z.

Stable ⟺ |zᵢ| < 1 pour TOUS les pôles zᵢ
Un pôle sur le cercle unité (|z| = 1) → le filtre est INSTABLE !

Méthode pour tester la stabilité de H(z)

H(z) = N(z) / D(z) = 1 / (1 + pz⁻¹ + qz⁻²) Étape 1 : Poser D(z) = 0 Multiplier par z² : z² + pz + q = 0 Étape 2 : Calculer Δ = p² - 4q Étape 3 : Si Δ > 0 → 2 pôles réels : z₁,₂ = (-p ± √Δ)/2 Si Δ < 0 → 2 pôles complexes conjugués : |z₁| = |z₂| = √q Étape 4 : Vérifier |zᵢ| < 1 pour tous les pôles

Raccourci pour pôles complexes

Si Δ < 0, les pôles sont complexes conjugués z₁,₂ = α ± jβ.
Leur module est |z| = √(α² + β²) = √q.

Stabilité : √q < 1 ⟺ q < 1

⚡ PIÈGES CLASSIQUES

❌ FAUX : "TF d'un signal échantillonné à T_e est T_e-périodique" → C'est (1/T_e)-périodique !

|z₂| = 0.3 < 1 ✅
Les deux pôles sont à l'intérieur du cercle unité, donc le filtre est stable.
